Gomoku Help Page

Gomoku Rules

Gomoku, also called Five in a Row, is a turn based game played by two players. Each player is assigned to a black or a white piece. The first player to get five pieces in row vertically, horizontally or diagonally is declared as the winner.

Main Page

The main page has two options that the user can interact with:

1- If an account exist, a user can login using their username and password. By using the login fields.

2- If an account does not exist. Then the user can create an account. By using the create account fields. And when doing so, the user must choose a username that is not on the server. Otherwise an account will not be created

Lobby Page

The lobby page has two options that the user can interact with:

1- A user has the choice of joining an existing public game, or they can join a private room that they have the key for.

2- If there are no rooms available or if the user wants to make a private room. They can do so, by clicking on create room. Which then will prompt the user to set a name for the room, and if they wish, they could also make it private by setting a password.

Game Page

When a player enters the game page, they will have to wait until another player. Once another player joins in, they can start playing the game by clicking on the crossed areas of the board.
